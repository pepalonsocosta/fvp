# FVP Client Commands

## `fvpc connect`

Connects to a VPN server.

```bash
fvpc connect --server 192.168.1.100:1194
```

## `fvpc disconnect`

Disconnects from the VPN server.

```bash
fvpc disconnect
```

_Note: Use Ctrl+C while connected to disconnect._

## `fvpc status`

Shows connection status and statistics.

```bash
fvpc status
```

_Note: Not yet implemented._

## `fvpc version`

Shows version information.

```bash
fvpc version
```

## `fvpc help`

Shows help message.

```bash
fvpc help
```
